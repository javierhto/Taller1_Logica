Para la elaboración de este taller, si hizo uso del lenguaje de programación Python, utilizando la librería PySwip.
Por lo cual se debe verificar tener la correcta instalación de estas librerías, como también asegurarse de tener swipl en el Path.

Sintomas: Puede realizar combinación de sintomas, donde se lista un total de 36. Si el/los sintoma/s ingresados se encuentran en más
una enfermedad, entonces el programa lista las posibles enfermedades que podría tener y sus sintomas.
